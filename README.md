# Executando o projeto localmente

### Instalação necessaria para rodar a aplicação.

- Docker
- Docker compose

### Para subir o projeto:

`docker-compose up -d`

## Para acessar a app:

`localhost:3000`

## Para rodar os testes:

`docker exec -ti web rspec`
